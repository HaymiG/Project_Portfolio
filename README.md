## Overview

This project is part of a structured portfolio build plan. The goal is to create a polished personal website that showcases:

- who you are
- what you can do
- projects you have built
- how people can contact you

## Project Status

Current progress:

- Base HTML boilerplate
- Global CSS reset and base styles
- Fixed navigation bar with hover effects

Planned sections:

- Hero
- About
- Skills
- Projects
- Contact
- Footer
- Responsive layouts (tablet and mobile)

## Tech Stack

- HTML5
- CSS3

## Getting Started

1. Clone the repository:

```bash
git clone https://github.com/HaymiG/portfolio.git
```

2. Open the project folder:

```bash
cd portfolio
```

3. Launch `index.html` in your browser.

## Customization

Update these parts first:

- `index.html`: your name, title, and section content
- `style.css`: color palette, spacing, and typography
- media assets: add your own icon, photos, and project visuals

## Roadmap

The build follows this workflow:

1. Setup and structure
2. Section-by-section HTML
3. Section-by-section CSS
4. Responsive design
5. Final polish and deployment
